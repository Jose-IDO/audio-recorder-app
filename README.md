# Audio Recorder App

A React Native application for recording and managing voice notes.

## Installation

1. Install dependencies:
```bash
npm install
```

2. For iOS:
```bash
cd ios && pod install && cd ..
```

3. Run the app:
```bash
npm run android
# or
npm run ios
```

## Features

- Record audio notes
- List all recordings
- Playback recordings
- Delete recordings
- Search functionality
